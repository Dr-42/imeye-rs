# Imeye-rs

Simple image viewer (Spiritual successor to [imeye](https://github.com/Dr-42/imeye) written in C)

## Building

```console
git clone https://github.com/Dr-42/imeye-rs
cd imeye-rs
cargo build --release
```

## Controls

| Key            | Action               |
| -------------- | -------------------- |
| Esc            | Close                |
| Left           | Previous image       |
| Right          | Next image           |
| Up             | Zoom in              |
| Down           | Zoom out             |
| F              | Toggle fullscreen    |
| W              | Move image up        |
| S              | Move image down      |
| A              | Move image left      |
| D              | Move image right     |
| Q              | Rotate anticlockwise |
| E              | Rotate clockwise     |
| R              | Reset view           |
| Click and drag | Pan image            |
| Scroll wheel   | Zoom in/out          |
